# Tornado smart contract data aggregator [![Build Status](https://github.com/esketit-cash/tornado-aggregator/workflows/build/badge.svg)](https://github.com/esketit-cash/tornado-aggregator/actions)

## Dependencies

1. node 12
2. yarn

## Start

```bash
$ yarn
$ cp .env.example .env
$ yarn test
```

## Deploying

Deploy to Kovan:

```bash
$ yarn deploy:kovan
```
